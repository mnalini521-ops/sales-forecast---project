# Sales Forecasting Project

This project focuses on sales forecasting using machine learning techniques.
The analysis and model development were done using Google Colab.

## Files Included
- sales_forecasting.ipynb : Main notebook with code and output
- data/ : Folder containing datasets used in this project

## Datasets Used
Only the following 3 datasets were used for this project:
- olist_orders_dataset.csv
-  olist_order_items_dataset.csv
-  olist_customers_dataset.csv

Other datasets were not used as they were not required for the analysis.

## How to Run
1. Open the notebook (sales_forecasting.ipynb)
2. Ensure datasets are present inside the data/ folder
3. Run all cells to reproduce the results

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Google Colab
